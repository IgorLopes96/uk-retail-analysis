## Online UK Retail Customer and Revenue Analysis

**Tools:** Python, SQL, Pandas, SQLite

### Project Overview
Analyzed a UK-based e-commerce transactional dataset to understand revenue trends,
customer concentration, and international performance.

### Key Analyses
- Month-over-month revenue growth using SQL window functions
- Top customers per country and global customer ranking using RANK()
- Data cleaning and exploratory analysis in Python
- Statistical analysis to identify purchasing patterns and seasonality

### Key Insights
- Revenue is highly concentrated among a small number of customers
- Strong seasonality with peak performance toward year-end
- Certain international markets generate disproportionate revenue
